# AI Resume Analyzer

## About This Project

I built this project to solve a simple but real problem - 
**how do you know if your resume actually matches a job description?**

This app takes a job description and a resume, analyzes them using AI, and gives a clear breakdown of:

* How well the resume matches the job
* What skills are missing
* Where the candidate is strong or weak
* A practical 4-week learning plan to improve
  

##  Tech Stack

* Python
* Streamlit
* OpenAI API


## What It Does

* Paste any job description
* Paste your resume
* Get an AI-powered analysis instantly

The app generates:

* ✅ Skill match percentage
* ❌ Missing skills
* 📊 Skill-by-skill comparison
* 📈 Proficiency ratings
* 🧠 Personalized learning roadmap
